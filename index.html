🚀 Mini App Setup

1. Deploy your app to a stable domain (e.g., your-domain.com)

2. Create manifest file at /.well-known/farcaster.json

3. Verify ownership using Warpcast Developer Tools

4. Share Mini App URL in Farcaster to auto-detect

📄 Required Manifest File

Create /.well-known/farcaster.json on your domain:

{
  "accountAssociation": {
    "header": "...",
    "payload": "...",
    "signature": "..."
  },
  "frame": {
    "version": "1",
    "name": "FC Photo Quest",
    "iconUrl": "https://your-domain.com/icon.png",
    "splashImageUrl": "https://your-domain.com/splash.png",
    "splashBackgroundColor": "#667eea",
    "homeUrl": "https://your-domain.com",
    "webhookUrl": "https://your-domain.com/api/webhook"
  }
}
